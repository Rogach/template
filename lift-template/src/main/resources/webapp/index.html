Main page

This is the handy template for Lift webapp project

Includes:

  • Jetty 9.0.4
  • Lift 2.5
    • #if includeConfigrity
  • Configrity 1.0.0
    • #fi #if includeScallop
  • Scallop 0.9.3
    • #fi
  • Bootstrap 3 RC1
  • JQuery 2.0.3
    • #if jsUnderscore
  • Underscore.js 1.5.1
    • #fi #if jsDojo
  • Dojo 1.9.1
    • #fi

Usage:

    #if pluginRevolver
  • Start server on port #{httpPort}: re-start
    • #fi #if pluginAssembly
  • Create a runnable jar: assembly
    • #fi #if pluginDeploy
  • Create a folder with files, ready for deployment: deploy-pack (the resulting files are located in target/deploy)
  • Deploy the project to server and start it (requires rsync and ssh): deploy-ssh "host" "path"
    • #fi
