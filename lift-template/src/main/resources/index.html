    #if pluginRevolver

  • Create a runnable jar: assembly

#if https

HTTPS keystore must be regenerated!

This template comes with pre-generated self-signed certificate. To ensure better security, you need to regenerate it using this command: 
keytool -keystore keystore -alias jetty -genkey -keyalg RSA
Or, even better, you can use entirely other certificate.
